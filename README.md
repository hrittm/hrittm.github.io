## Run Locally

**Prerequisites:**  Node.js

Clone:
```sh-session
git clone https://github.com/hrittm/hrittick-web.git
cd hrittick-web
```


Install dependencies:
```sh-session
npm install
```

Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key

Run the app:
```sh-session
npm run dev
```
